阿里云CentOS7.0上搭建FTP服务器

判断是否安装了vsftpd

rpm -qa | grep vsftpd

如果没有安装则安装vsftpd

yum -y install vsftpd

查看服务的状态，没运行则执行启动命令

systemctl status vsftpd 查看服务的状态
systemctl start vsftpd  启动服务
systemctl stop vsftpd   停止服务
systemctl restart vsftpd    重启启动服务
systemctl enable vsftpd  开机自启动
systemctl disable vsftpd   取消开机自启动

修改配置文件

cd /etc/vsftpd/
vim vsftpd.conf
anonymous_enable=NO

创建用户及密码

useradd user1 -d /home/ftp1 创建用户
passwd user1 设置密码

阿里云防火墙开一下端口号
